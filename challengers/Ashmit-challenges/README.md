# Ashmit's Daily Dev Challenges

A collection of solutions for daily development challenges including frontend and backend tasks.

## Challenges

### Easy Tasks
- **Easy Task 1**: Build to-do app - A fully functional todo application with local storage
- **Easy Task 2**: Create REST API - A working REST API implementation using Node.js

### Medium Tasks
- **Medium Task 3**: Build responsive footer - A responsive footer that works on all screen sizes
- **Medium Task 4**: Create modal popup - A modal popup with animations and interactions

### Hard Tasks
- **Hard Task 5**: Build image gallery - An interactive image gallery with lightbox functionality

### Additional
- **Leaderboard**: Display top performers with scores

## How to Use

Each folder contains `index.html`, `script.js`, and `styles.css` files. Open the `index.html` in your browser to view the challenge.

## Features

- Modern, responsive design
- Clean, readable code
- Interactive user experiences
- Local storage integration (where applicable)
- Cross-browser compatibility
