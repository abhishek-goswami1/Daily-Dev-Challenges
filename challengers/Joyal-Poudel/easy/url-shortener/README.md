# Day 7 - EASY Task 1: URL Shortener Logic

## Description
A beautifully designed URL shortener interface built with vanilla HTML, CSS, and modern JavaScript. Features a glassmorphism design system requested for premium visual aesthetics.

### Features
- 🎨 **Premium UI**: Glassmorphism, smooth animations, and dark mode aesthetic.
- 🔗 **Local Logic**: Simulates a backend shortening process by securely storing links in `localStorage`.
- 📋 **Copy to Clipboard**: Instant feedback interactions.
- 🗑️ **Link Management**: Delete single links or clear all history.

## Technologies Used
- HTML5
- CSS3 (Variables, Flexbox, Animations, Glassmorphism)
- Vanilla JavaScript (LocalStorage)
- FontAwesome Icons

## Deployment
Can be run by opening `index.html` in any modern web browser.
