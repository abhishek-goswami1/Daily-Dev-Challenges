# Day 7 - EASY Task 2: Debounce Function

## Description
A practical demonstration of the JavaScript `debounce` technique. This project simulates an API-driven search interface where keystrokes are registered instantly, but the actual "API query" is delayed until the user stops typing, saving resources and network calls.

### Features
- ⚡ **Debounce Logic**: Custom implementation of `debounce()` preventing excessive function calls.
- 📊 **Live Analytics**: A dashboard showing keystroke count vs simulated API calls to prove the debounce efficiency.
- 🎨 **Premium UI**: Glassmorphism, animated loading spinners, and modern aesthetic.

## Technologies Used
- HTML5
- CSS3
- Advanced Vanilla JavaScript (Closures, Promises, Timeouts)

## Deployment
Can be run by opening `index.html` in any browser.
