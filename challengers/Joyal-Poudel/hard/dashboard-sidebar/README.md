# Day 7 - HARD Task 5: Responsive Dashboard Sidebar

## Description
A fully responsive, production-ready Admin Dashboard Sidebar. Built thoughtfully with complex states including Desktop Collapse, Mobile Off-canvas, and nested sub-menu accordions.

### Features
- 📱 **Fully Responsive**: Changes behavior entirely on mobile (transforms into an absolute off-canvas drawer with an overlay backdrop).
- ↔️ **Desktop Collapse**: Clicking the expand/collapse button smoothly shrinks the sidebar to an icon-only view.
- 🗂️ **Nested Submenus**: Accordion-style sub-menus that automatically manage their open/closed states.
- 🎨 **Premium Aesthetic**: Detailed dark theme, glassmorphism cards in the main section, and high-quality `Phosphor Icons`.
- ✨ **Attention to Detail**: Custom scrollbars, transition timings, active states, and hover effects.

## Technologies Used
- HTML5 (Semantic Structure)
- CSS3 (Flexbox/Grid, CSS Variables, Transitions)
- Vanilla JavaScript (Event Listeners, DOM Class Manipulation)
- Phosphor Icons via CDN

## Deployment
Open `index.html` in your browser. Resize the window to see how it reacts to mobile vs desktop sizes.
