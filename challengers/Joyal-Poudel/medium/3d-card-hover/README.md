# Day 7 - MEDIUM Task 3: 3D Card Hover Effect

## Description
A premium interactions project demonstrating how to build a 3D tilting card that responds to mouse movement using vanilla JavaScript and CSS 3D transforms.

### Features
- 🚀 **Interactive 3D**: Tracks mouse parameters and maps them to `rotateX` and `rotateY` transforms.
- ✨ **Dynamic Glare**: A lighting reflection that follows your cursor, creating a realistic material feel.
- 📐 **Depth Layers**: Child elements use `translateZ` to pop out from the background while rotating.
- 🎨 **Glassmorphism**: Built on top of a dark-mode frosted glass effect.

## Technologies Used
- HTML5
- CSS3 (perspective, transform-style: preserve-3d, backfliter)
- Vanilla JavaScript (BoundingClientRect math)

## Deployment
Can be run by opening `index.html` in any browser. Works best on desktop cursors.
