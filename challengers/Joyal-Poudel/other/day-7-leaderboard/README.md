# Day 7 - 🏆 Leaderboard

## Description
A visually stunning and fully dynamic leaderboard for Day 7 of the Daily Dev Challenges. Renders challenger data with animated podium positions, score bars, and task difficulty badges.

### Features
- 🥇 **Animated Podium**: Top 3 challengers are rendered in the classic 2nd/1st/3rd podium layout with staggered animations.
- 📊 **Animated Score Bars**: Each challenger has a bar that fills to indicate their relative score.
- 🏷️ **Task Badges**: Color-coded difficulty tags (Easy / Medium / Hard) to show what each person completed.
- 🟢 **Live Indicator**: Pulsing "Live" badge for a real-time feel.
- ✨ **Staggered Animations**: Cards rise and slide in sequentially for a premium entrance effect.

## Technologies Used
- HTML5
- CSS3 (Gradients, Keyframe Animations, CSS Variables)
- Vanilla JavaScript (Dynamic DOM Rendering, Data Sorting)

## Deployment
Open `index.html` in any modern browser.
